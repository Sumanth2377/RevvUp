# **App Name**: AutoMind

## Core Features:

- User Registration & Vehicle Profiles: Secure user accounts to manage multiple vehicle profiles including details like make, model, year and mileage.
- Maintenance Schedule Setup: Define and customize maintenance tasks for each vehicle, setting intervals based on time or distance.
- Intelligent Due Date Calculation: Automatically calculates upcoming due dates for maintenance tasks based on set intervals and service history. Generative AI acts as a tool suggesting optimal maintenance schedules based on vehicle usage patterns and manufacturer recommendations.
- Automated Reminders & Notifications: Configurable reminders sent via email, SMS, or push notifications before a scheduled service is due.
- Service History Log: Record completed maintenance with date, mileage, and notes for each vehicle, displaying historical data.
- Dashboard Overview: Visual dashboard providing an overview of all vehicles, upcoming services, and overdue tasks.
- Database Integration: Persistent storage of user data, vehicle info and maintenance logs

## Style Guidelines:

- Primary color: Dark blue (#2E3192) for trustworthiness and reliability.
- Background color: Light gray (#F0F2F5) to provide a clean and modern feel.
- Accent color: Orange (#E67E22) to highlight important CTAs and alerts.
- Body font: 'PT Sans', a humanist sans-serif font, is chosen to give a combination of a modern and warm feel to the site.
- Headline font: 'Space Grotesk', a proportional sans-serif is selected to create a contemporary high-tech vibe for headers and titles.
- Use clear, recognizable icons for vehicle types, maintenance tasks and alert types.
- Subtle transitions and loading animations to improve user experience.
